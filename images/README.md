# images

放圖片的地方。網頁會自動去找以下檔名 —— **檔案不存在時會優雅地退回成漸層色塊，不會出現破圖**，所以你可以慢慢補。

| 檔名 | 用在哪裡 | 建議尺寸 |
|---|---|---|
| `portrait.jpg` | 首頁「簡介」章節 | 直式，約 1000 × 1250 |
| `research.jpg` | 首頁「研究」章節 | 直式，約 1000 × 1250 |
| `publications.jpg` | 首頁「著作」章節 | 直式，約 1000 × 1250 |
| `teaching.jpg` | 首頁「教學」章節 | 直式，約 1000 × 1250 |

小提醒：

- 版面用 `object-fit: cover` 裁切，比例 4:5（桌機）／16:11（手機），主體請放中間。
- 圖片上會壓一層由下往上的暗色漸層，所以下緣不要放重要細節。
- 上傳前壓到 300 KB 以內，載入會快很多。
